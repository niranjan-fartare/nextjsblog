## Bandwidth Statistics
You can retrieve combined statistics for all projects hosted on a particular server by making a GET request to any [Arch Linux Mirror](#arch-linux) with ` /stats ` appended to the URL.
For example, to get the stats for the India server:

```curl
curl https://in.arch.niranjan.co/stats
```
